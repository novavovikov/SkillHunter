# SkillHunter
SkillHunter project

## Сборка образов
```sh
docker-compose build
```

## Запуск проекта
```sh
docker-compose up
```

## backend 
[Framework nest](https://nestjs.com/)

### Запуск команд
переход в терминал образа
```sh
docker-compose run backend sh
```
затем доступны команды, например форматирование
```sh
npm run format
```
